Privacidade

Este site usa cookies para funcionalidades e coleta dados anonimizados para gerar estatísticas de tráfego conforme política de privacidade do Google Analytics. Em nenhuma hipótese este site coleta informações sobre seus usuários para fins além dos aqui explicitados.

Quais dados são coletados pelo serviço de estatística de dados do Google Analytics?

  • o tempo total que um usuário passa no site;
  • o tempo que um usuário passa em cada página e em que ordem essas páginas foram acessadas;
  • em quais links internos ele clicou (com base no URL da próxima visualização de página).
  • a localização geográfica do usuário;
  • o navegador e o sistema operacional usados;
  • o tamanho da tela e se o Flash ou o Java está instalado;
  • o site de referência.
